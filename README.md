# EchoFrame Cast – AI Generated Podcast

This project was created as part of a challenge from the DIO platform focused on the use of Generative AI to create podcast content.

The objective was to use artificial intelligence tools to generate the script and voice narration for a short podcast episode.

## Podcast Theme

This episode explores how artificial intelligence can assist creators in developing cinematic storytelling and machinima projects inside video games.

The creative concept is inspired by the EchoFrame universe, where storytelling, machinima, and technology intersect.

## Tools Used

- ChatGPT – script generation
- AI Text-to-Speech – voice narration
- Basic audio editing tools
- Spotify for Podcasters – podcast publishing

## Project Structure

audio/
Final podcast audio file.

prompts/
Prompts used to generate the script and creative direction.

roteiro/
Podcast script used to generate the narration.
